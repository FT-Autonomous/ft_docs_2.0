## How to add a new year

1. Save headshots in `docs/media/hof/` ‚Äî use clear names like `2026-firstname-lastname.jpg`.
2. Duplicate a year section and update:
   - The year in the `##` heading
   - Each `<img src="media/hof/‚Ä¶">` path
   - Names and roles inside `<figcaption>‚Ä¶</figcaption>`

> üîé Accessibility: keep descriptive `alt="‚Ä¶"` text on each photo.
